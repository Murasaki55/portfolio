  • AC14.02 | Visualiser des données

Durant ma 1ère année de BUT Informatique, j'ai eu l'opportunité de mettre en place mon apprentissage dans la conception d'une base de données de l'entreprise fictive : « Matski »

Partie 1 : Création de la base de données

Base de données actuel de Matski

Base de données future

Matski m'as fourni leur base de données actuel, une base incomplète avec des erreurs. Après avoir expliqué au client se qu'il n'allait pas dans leur base, j'en ai réaliser une autre en fonction de leur nouveau besoin.

    Leur base actuel a les erreurs suivante :
  • La table Client possède des attributs qui ne sont pas utiles
  • La table Articles possède des attributs "Contact" devrait être dans la table Client.
  • La table Article possede l'attribut PrixVente également présent dans l'association TarifVente, l'attribut doit donc être supprimé de Article.

La nouvelle base que j'ai réaliser règle les erreurs mais également ajoute une partie achat de l'entreprise. lle ajoute les tables suivantes : Fournisseur, Contact, Approvisionnement et Caractéristique d'un article

Partie 2 : Requêtage

Durant notre étude de la table l’entreprise nous as informer qu’il voudrait connaître quelques informations sur la base de données pour faire une analyse des données économique de l’entreprise, ils nous ont donc posée plusieurs questions que nous avons répondue en faisant des requête sur leur base de données actuel.
