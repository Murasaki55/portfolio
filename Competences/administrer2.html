    Apprentissages critiques :

Durant ma 2ème année de BUT Informatique, j'ai eu l'opportunité de mettre en place mon apprentissage dans un groupe de 5 étudiants dans la conception et le développement d'un site e-commerce de vente de vélos électriques d'occasion pour le client fictif Upway

Le projet réalisé repose sur des architectures communicantes, notamment des applications web utilisant des échanges client–serveur. Nous avons développé des applications capables de communiquer entre une API Rest en C# et un client en VueJs, en respectant les principes du modèle client–serveur.

Partie 2 : Serveurs

Dans le cadre du projet, l’application a été hébergée sur une infrastructure cloud fournie par Microsoft Azure. Cette solution nous a permis de disposer d’un environnement distant accessible à tout moment. L’utilisation d’Azure a facilité le déploiement et les tests de l’application.

Partie 3 : Sécuriser les services

Toute la partie sécurisation des données a été effectuée, le serveur Azure n'était accessible que par les 5 personnes travaillant sur le projet, les comptes étaient protégés par des mots de passe différents. L'accès aux différentes pages de l'application a été réglementé par des comptes avec des droits d'accès différents. La base de données est sécurisé par un mot de passe hashé.
