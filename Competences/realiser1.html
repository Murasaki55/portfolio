Compétence : Réaliser un développement d’application (Niveau 1)

Durant ma 1ère de BUT Informatique, j'ai eu l'opportunité de mettre en place mon apprentissage dans un groupe de 3 étudiant dans la conception et le développement d'une application de réservation de véhicule pour l'enseigne française de grande distribution : « Intermarché »

Partie 1 : Conception

Nous avons commencé ce projet par une approche de conception pour orienter notre programmation future. Nous avons utilisé Visual Paradigme pour la création d'un diagramme de classe pour se représenter les objets qui seront manipulés dans le code ainsi que des diagrammes de séquences pour représenter les actions du client et les réponses du système.

Ces diagrammes nous on permis de nous projeter dans la programmation dans laquel nous aurons 10 classe pour manipuler nos objets, mais également mettre en mumière les interfaces utilisateurs avec les réponses du code

Partie 2 : Interface utilisateurs

Avant de passer au développement, nous avons réaliser des maquettes pour l'iterface et les diiférents fenêtre visible par l'utilisateur. Nous avons mis au point cette maquette en WPF dans Visual Studio.

La maquette suis toutes les étapes de la réservation d'un véhicule. L'application sera utilisé par des salarié de l'entreprise donc la réservation passe tout d'abord par une authentification, elle passe par une recherche de véhicule dans les différentes agences, la diponibilité d'un véhicule, ajout des options, et l'affectation à un client en passant par la création du client si il n'existe pas dans la base de données du logiciel.

Partie 3 : Développement

A partir de la conception et des maquettes, nous avons commencer le développement. Pour la programmation nous avons utilisé WPF C#. Nous avons créer les classes des objets qui ont été décrit par le diagramme de classe. Nous avons coder les fonctions du code et nous les avons lié avec l'interface de l'application. Pour chaque fonction des tests manuelle on été réalisé afin de validé leur fonctionnement.
